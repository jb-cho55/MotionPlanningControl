# CLAUDE.md

## Project Context
이 프로젝트는 ADAS / 자율주행 / 차량 제어 / 검증 관련 프로젝트이다.
주요 작업은 MATLAB/Simulink, Python, C++ 코드 분석 및 수정이다.

Claude는 코드를 수정하기 전에 기존 구조, 함수 입출력, 테스트 조건을 먼저 파악해야 한다.
추측으로 대규모 리팩터링하지 말고, 최소 변경으로 문제를 해결한다.

## Response Rules
- 기본 답변은 한국어로 한다.
- 코드 설명은 초보자도 이해할 수 있게 단계적으로 설명한다.
- 수정한 코드는 왜 수정했는지 함께 설명한다.
- 불확실한 부분은 확정적으로 말하지 말고 확인이 필요한 부분으로 표시한다.
- 긴 설명보다 바로 적용 가능한 코드와 명령어를 우선한다.

## Coding Rules
- 기존 함수 이름, 입출력 형식, 파일 구조를 임의로 바꾸지 않는다.
- MATLAB Function Block / Simulink 코드에서는 codegen 호환성을 고려한다.
- Simulink용 MATLAB 코드는 동적 배열, 지원되지 않는 함수 사용을 피한다.
- Python 코드는 타입 힌트와 간단한 예외 처리를 우선한다.
- 변경 전후 차이를 짧게 요약한다.

## Verification Rules
코드 수정 후 가능한 경우 아래 순서로 검증한다.

1. 문법 오류 확인
2. 간단한 입력 예제로 동작 확인
3. 기존 테스트가 있으면 테스트 실행
4. 변경으로 인해 영향받는 함수 확인

검증을 실행하지 못한 경우, 실행하지 못한 이유와 사용자가 직접 실행할 명령어를 알려준다.
